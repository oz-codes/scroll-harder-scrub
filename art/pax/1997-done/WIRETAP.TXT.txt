15,01        F4 - Color mode (Color/iCE Color/Greyscale/No Color/Hidden Text)
15,01        F5 - Scrolling (Smooth/Normal/Baud Rates)
15,01        F6 - File listing mask

15,01        ALT-D - Change Directory
15,01        ALT-C - Copy File
