15,01==+ What's New +=============================================================
15,01  +============+


15,01--+ Version 1.1 --- Released on : April 1st, 1994 +--------------------------

15,01 (Viewer) - Reduced memory requirement by 50k
15,01 (RIP)    - Added bezier support.
15,01 (DOC)    - Added a NFO file with viewer.
15,01 (Viewer) - Fixed error with long artist names.
15,01 (Viewer) - Fixed spelling error in CC's name.
15,01 (RIP)    - Fixed bug which improperly viewed some RIP files.
15,01 (RIP)    - Removed sound when finished viewing RIPs.

15,01--+ Version 1.0 --- Released on : March 1st, 1994 +--------------------------


15,01 (VIEWER) - First official release of Gothic View to the public.


15,01  +==============+
15,01==+ Requirements +===========================================================
15,01  +==============+

15,01   - 80286 processor or better.
15,01   - 300k of free memory.
15,01   - Either a CGA,EGA,VGA or better graphics card.
15,01   - Dos v3.1 or better.
