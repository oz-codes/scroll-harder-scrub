15,01HAPPY 1st ANNIVERSARY TO DARK! Februrary 1995 starts off with a bang! Yes, it
15,01IS Darks first anniversary. And to celebrate this occasion DarkView 3.1 has
15,01been released which has been improved by a great deal since the last version.
15,01I am also happy to announce the addition of Sudden Impact and Epitaph to code
15,01DarkView and take some pressure off me. The improvements in this version were
15,01the ideas of both these renowned coders. The new INI configuration has been
15,01coded by Epitaph whereas Sudden Impact helped me fix some minor bugs and some
15,01major bugs here and there. Sudden Impact also coded the BIN viewer. The
15,01following has been added to DarkView 3.1:

15,01 VGA/SVGA viewing modes for ansi(s)
15,01 INI file configuration for DarkView with Interface
15,01 BIN support
15,01 Improved speed of GIF and PCX viewing
15,01 Command for switching interfaces
15,01 Some bug fixes here and there


15,01DarkView 3.1 *REQUIREMENTS*
15,01 # 386SX or higher HIGHLY recommended
15,01 # 500K+ of conventional memory recommended
15,01 # VGA Card very HIGHLY recommended (SVGA card required for Hi-res VGA's etc.)
15,01 # Approximately 140K of Hard Drive Space
15,01 # DARKVIEW.EXE REQUIRED :)
